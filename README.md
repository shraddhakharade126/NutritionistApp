# Personal Nutritionist App

A complete Android application built with Java + SQLite.

## Features
- Register / Login with SQLite authentication
- Food Nutrition Search (25+ foods built-in)
- BMI Calculator & Daily Calorie estimator
- Diet Plan with meal reminders
- Recipe Manager (add & view recipes)
- Profile management

## Setup Instructions

1. Open **Android Studio**
2. Click **Open** and select this `NutritionistApp` folder
3. Wait for Gradle to sync (first time may take a few minutes)
4. Connect Android phone (enable USB Debugging) OR use AVD Emulator
5. Click the ▶ **Run** button

## Requirements
- Android Studio Hedgehog or newer
- Java 8+
- Min Android SDK: 21 (Android 5.0 Lollipop)
- Target SDK: 34 (Android 14)

## Package
com.nutritionist.app
